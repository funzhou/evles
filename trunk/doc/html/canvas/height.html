height(value)

概述

canvas方法之一。 为canvas对象设置高度

参数:value

高度的数值

描述:

为canvas对象设置高度

HTML 代码:

<canvas id="test" width="507px" height="320px" style="border:1px solid #F00">

elves 代码:

$(function() {   
	$("#test").canvas("desk")    //初始化canvas
    $("$desk").height(500);  //选中刚刚初始化的canvas对象，将高度设置为500
});
